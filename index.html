HTML and CSS

HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are the foundational building blocks of web development. HTML defines the structure of the webpage, while CSS handles the layout and design.

<div> <header> <footer>
  • HTML creates the structure of the webpage.
  • CSS adds styling and layout.
  • HTML is essential for web pages to work.
JavaScript

JavaScript is a programming language that allows you to implement dynamic features on web pages. It can be used for things like form validation, creating interactive elements, and much more.

let x = 10; document.getElementById("id"); function myFunction() {}
  • JavaScript makes your webpage interactive.
  • It's essential for dynamic content.
  • JavaScript integrates with HTML and CSS.
DOM Manipulation

DOM Manipulation refers to the ability to interact with the Document Object Model (DOM) in JavaScript. This allows you to change the structure and content of your web page dynamically.

document.getElementById() document.createElement() document.querySelector()
  • DOM Manipulation allows dynamic page changes.
  • It is a core part of JavaScript web development.
  • It interacts with HTML elements via JavaScript.
APIs

APIs (Application Programming Interfaces) allow different software systems to communicate with each other. In web development, APIs are used to fetch data from remote servers, such as in the case of weather apps or online databases.

fetch(url) axios.get(url) JSON.parse()
  • APIs allow the web to interact with external services.
  • They are widely used in modern web applications.
  • APIs can fetch real-time data to display on your site.
Web Design

Web Design involves the planning and creation of websites. It covers areas such as layout, color scheme, typography, and user experience (UX).

flexbox grid layout @media queries
  • Responsive design ensures your website looks good on any device.
  • Web design focuses on user experience.
  • CSS frameworks like Bootstrap can speed up development.
